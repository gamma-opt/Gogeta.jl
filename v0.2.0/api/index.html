Public API

These are all of the functions and data structures that the user needs to know in order to use this package.

Tree ensembles

Data structures

  • TEModel - holds the parameters from a tree ensemble model

Tree model parameter extraction

  • extract_evotrees_info - get the necessary parameters from an EvoTrees model

MIP formulation

  • TE_formulate! - formulate a JuMP model from a tree ensemble without the split constraints

Input optimization

  • add_split_constraints! - add all split constraints to the formulation
  • tree_callback_algorithm - used to add only necessary split constraints during callbacks

Showing the solution

  • get_solution - get human-readable solution in the form of upper and lower bounds for the input variables

Neural networks

MIP formulation

  • NN_formulate! - formulate a JuMP model, perform simultaneous bound tightening and possibly compression

Compression

  • NN_compress - compress a neural network using precomputed activation bounds

Forward pass

  • forward_pass! - fix the input variables and optimize the model to get the output

Sampling-based optimization

  • optimize_by_sampling! - optimize the JuMP model by using a sampling-based approach
  • optimize_by_walking! - optimize the JuMP model by using a more sophisticated sampling-based approach

Convolutional neural networks

Data structures

  • CNNStructure - container for the layer stucture of a convolutional neural network model

Parameter extraction

  • get_structure - get layer structure from a convolutional neural network model

MIP formulation

  • CNN_formulate! - formulate a JuMP model from the CNN

Forward pass

  • image_pass! - fix the input variables and optimize the model to get the ouput

Sampling-based optimization

  • optimize_by_walking_CNN! - optimize the JuMP model by using a more sophisticated sampling-based approach
